# AIM: Creating a student database 
- Student details along with their course details given in csv files
- Used SQL and BASH to transfer this data to a database
- Used relational databases to organise the data into tables, connecting them with foreign keys using the major_ids and course_ids
- Used a junction table `majors_courses` to implement many-to-many relations
