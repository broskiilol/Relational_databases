# Bike rental - Workshop
###  This workshop is about creating a bike rental system using SQL and bash

## Features:
- Using a menu-driven system for different operations.
- Operations include- Renting bike, Returning bike, exiting
- Maintaining a `customers` database to keep track of their names and phone numbers
- Maintaining a `bikes` database to keep track of available bikes
- Creating new customers as more people rent out bikes, and storing it in a customer database using their unique phone numbers
- Maintaining a `rentals` database to keep track of rent date, and return date
